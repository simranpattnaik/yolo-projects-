 YOLO Computer Vision Projects

This repository contains multiple **computer vision projects using YOLO and OpenCV**.  
The projects demonstrate different real-time detection tasks such as **object detection, segmentation, and pose estimation**.

These projects are implemented in **Python** and use modern **deep learning and computer vision techniques**.

---

# 📂 Project Files

| File | Description |
|-----|-------------|
| `object_detection.py` | Detects objects in images or video using YOLO. |
| `object_segmentation.py` | Performs object segmentation to separate objects from the background. |
| `pose_estimation.py` | Detects and tracks human body keypoints using pose estimation. |
| `yolo_run.ipynb` | Jupyter notebook for running YOLO models and experiments. |
| `yolo.py` | Main YOLO configuration and helper functions. |
